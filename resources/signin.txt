##Welcome to the Google Maps Engine API tutorial!##  
This tutorial will take you through getting set up as a Google developer and beginning to manage your data and interact with Google Maps Engine using the API. You will learn how to read public data from a Google Maps Engine project as well as create your own personal project and add data to it.

It seems you have already started managing your Maps Engine data in this tutorial. If you are ready to resume your work, click the button below to sign in and get going again!
